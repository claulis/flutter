# Principais comandos do Flutter:

1. **`flutter create <nome_do_projeto>`**: Cria um novo projeto Flutter com a estrutura básica de diretórios e arquivos.

2. **`flutter run`**: Compila e executa o aplicativo em um dispositivo conectado ou emulador.

3. **`flutter build <target>`**: Gera o aplicativo em um formato específico para distribuição (por exemplo, `apk` para Android ou `ipa` para iOS).

4. **`flutter pub get`**: Baixa todas as dependências listadas no arquivo `pubspec.yaml` do projeto.

5. **`flutter pub upgrade`**: Atualiza todas as dependências para a versão mais recente permitida pelas restrições no arquivo `pubspec.yaml`.

6. **`flutter pub outdated`**: Mostra as dependências que estão desatualizadas no projeto.

7. **`flutter doctor`**: Verifica o ambiente de desenvolvimento Flutter e fornece um relatório de diagnósticos para ajudar a resolver possíveis problemas.

8. **`flutter clean`**: Remove arquivos temporários e de build do projeto, útil para resolver problemas de cache.

9. **`flutter devices`**: Lista todos os dispositivos conectados e emuladores disponíveis para execução do aplicativo.

10. **`flutter test`**: Executa os testes de unidade e de integração do projeto.

11. **`flutter analyze`**: Analisa o código fonte do projeto para identificar possíveis problemas de estilo e erros.

12. **`flutter install`**: Instala a aplicação no dispositivo conectado sem executá-la.

13. **`flutter upgrade`**: Atualiza o Flutter para a versão mais recente.

14. **`flutter format <caminho_arquivo>`**: Formata o código Dart no caminho especificado de acordo com as diretrizes de estilo do Flutter.

15. **`flutter gen-l10n`**: Gera arquivos de localizações baseados nas definições no diretório `lib/l10n`.
